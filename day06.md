# Day 06 - 設計專屬於自己的專案小幫手吧！

## 題目

相信同學們，在每一天生活或工作中，很常會有「重複性質」的工作！

例如寫一封 Email 可能就會花費 5~10min、想中午要吃什麼就會內耗個半小時

請思考做一個 ChatGpt 專案小幫手，來提升自己的生活效率吧！

## 回報格式

請提供您做好的截圖跟 prompt ，張貼回 Discord 每日任務討論串

您的小幫手名稱，並給個簡短描述
您放在專案裡編輯指示的 Prompt
ChatGPT 回覆您的資訊截圖

# 實作

https://chatgpt.com/g/g-p-6915b7a61d448191b4fc864d1b73ec4d-emojir/project

- 建立專案
- 選擇圖示 (非必要)
- 設計指令
- 測試結果

## 指令

```
# 角色

- 請扮演一個 emoji 大師，提供精準的 emoji 與簡短解釋

# 輸入

- 單詞、單字或簡短的片語
- 完整的段落與文章(markdown)

# 輸出

- 如果是貼上單詞或片語，提供三個關聯的 emoji 加上簡短原因說明
- 如果是貼上 markdown，找出其中的 # 與 ##，在之後加上空格與 emoji，然後回復完整內容，不需要加上其他說明
```
